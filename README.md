# kaleab

Kaleab S. Melkie's official website

## Setup

1. Copy `.env.example` to a new `.env` file, and configure it as needed

2. `yarn install`

## Scripts

```bash
// development:
yarn dev

// production:
yarn prod
yarn start
```
